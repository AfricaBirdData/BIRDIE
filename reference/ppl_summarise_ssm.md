# Summarise predictions from state-space JAGS model

Summarise predictions from state-space JAGS model

## Usage

``` r
ppl_summarise_ssm(fit, counts, sp_code, linear = TRUE, config, ...)
```

## Arguments

- fit:

  A model fit generated by
  [`ppl_fit_ssm_model`](https://africabirddata.github.io/BIRDIE/reference/ppl_fit_ssm_model.md)

- counts:

  A data frame with counts generated by
  [`ppl_create_data_ssm`](https://africabirddata.github.io/BIRDIE/reference/ppl_create_data_ssm.md)

- sp_code:

  SAFRING code of the species of interest

- linear:

  If TRUE (default) then summaries are given the linear scale. If FALSE,
  then summaries are given in the modelling scale (typically log scale)

- config:

  A list with pipeline configuration parameters. See
  [`configPipeline`](https://africabirddata.github.io/BIRDIE/reference/configPipeline.md)

- ...:

  Other parameters to pass on to
  [`prepGEECatchmData`](https://africabirddata.github.io/BIRDIE/reference/prepGEECatchmData.md)
