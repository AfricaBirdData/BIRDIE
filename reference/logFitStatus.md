# Generate fit status messages

Generate fit status messages

## Usage

``` r
logFitStatus(.fit_out, .year, .sp_code, .config)
```

## Arguments

- .fit_out:

  Fit status object generated by
  [`ppl_fit_occu_model`](https://africabirddata.github.io/BIRDIE/reference/ppl_fit_occu_model.md)

- .year:

  Year to run to the pipeline for

- .sp_code:

  Species SAFRING code

- .config:

  Config object from
  [`configPipeline`](https://africabirddata.github.io/BIRDIE/reference/configPipeline.md)
