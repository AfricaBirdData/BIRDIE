# Create indicator storage files

Create indicator storage files

## Usage

``` r
ppl_create_indtr_file(sp_code, year, overwrite_indtr)
```

## Arguments

- sp_code:

  SAFRING code of the species to run the pipeline for

- year:

  Year to run to the pipeline for

- overwrite_indtr:

  Logical. If TRUE, existing files in directories corresponding to the
  species in config\$species will be overwritten.
